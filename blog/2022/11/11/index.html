RayGun4Net Not Showing Errors

and there is no error to debug.
Created at: 11/11/2022 11:02:27

If you are having trouble with error not showing up in RayGun, and you are using the RayGun4Net Nuget package, set `throwOnError=true` in the web.config. 

                <RaygunSettings apikey="[Raygun4Net api key goes here]" throwOnError="true"/ >

In my case, the recommended tip to add the following to the global.asax did the trick. Note, that this line should go in the `Application_Start` method. 

                ServicePointManager.SecurityProtocol = SecurityProtocolType.Tls11 | SecurityProtocolType.Tls12

Source: https://github.com/MindscapeHQ/raygun4net#troubleshooting
