About Charlor

                
            

            

                
Charlor = Char + Color. A zero-player grid toy where colored
                    characters (“globs”) write into (row, column) cells.

                
    
                    
  • Char match: If an incoming char hits the same char in a cell, that cell’s
                        neighbors are cleared and disabled.
    • 
                    
  • Color match: If the incoming color is near the existing color, the center +
                        neighbors are tinted and disabled.
    • 
                    
  • 4 vs 8 neighbors: Toggle which neighbor function you call to change spread.
                    
    • 
                    
  • Hard Block: When ON, disabled cells can never accept writes.
