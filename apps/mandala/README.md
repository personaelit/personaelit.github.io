# Mandala Generator

A single-file, printable mandala coloring page generator. No build step, no dependencies.

## Features

**Controls**
- **Symmetry** — 6, 8, 10, 12, 16, or 24-fold rotational symmetry
- **Rings** — 3–10 concentric rings
- **Line Weight** — Thin, Medium, or Thick strokes
- **Style** — Any (mixed), Floral, or Geometric pattern pool
- **Ring dividers** — optional concentric circle outlines between rings
- **Radial guides** — optional spoke lines at segment boundaries

**Patterns**
- Floral: petal, lotus, teardrop, droplet, nested petal, circle
- Geometric: diamond, spike, fan, arch, chevron, hexagon

**Actions**
- **Randomize** — generates a new mandala; seed is displayed for reproducibility
- **Download PNG** — exports a high-resolution PNG (2× screen size)
- **Print** — hides controls and sends the mandala to the printer (also works as Print to PDF)

## Usage

Open `index.html` directly in a browser — no server required.

For the main site, visit `/apps/mandala/` or link from `content/digital-craft.html`.
