# Zero Sum

Zero Sum — a grid puzzle where each row and column must sum to exactly zero. You're given a partially filled grid of positive and negative integers, and you drag number tiles into empty cells to balance everything out. Harder levels have fewer givens, larger grids, or constrained tile sets where you must choose wisely — because the tiles are shared across rows and columns.

The core mechanic is satisfying: every number you place ripples across two axes at once.
